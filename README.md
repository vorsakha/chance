# ✳️ CHANCE

Chance grabs brazilian serie A (Brasileirão) betting odds from several websites and displays per match, so people get an easy understanding of the current matches odds.

## ✔️ Tech

- Nextjs
- Puppeteer
- Typescript
- Styled Components

## ✔️Install

```
yarn install
```

### ✔️ Develop

```
yarn dev
```
